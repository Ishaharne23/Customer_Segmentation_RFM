# Customer Segmentation using RFM Analysis

## Project Overview
This project performs Customer Segmentation using RFM (Recency, Frequency, Monetary) Analysis on the Online Retail Dataset.

## Objective
To identify valuable customers and group them into meaningful segments for targeted marketing strategies.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Online Retail Dataset

## Methodology
1. Data Cleaning
2. RFM Metric Calculation
3. RFM Scoring
4. Customer Segmentation
5. Data Visualization

## Customer Segments Identified
- Champions
- Loyal Customers
- Potential Loyalists
- At Risk Customers
- Lost Customers

## Key Findings
- Lost Customers formed the largest customer segment.
- Loyal Customers contributed significantly to repeat purchases.
- Champions represented the highest-value customers.
- Customer segmentation can help businesses improve retention and marketing effectiveness.

## Business Recommendations
- Reward Champions with loyalty programs.
- Retarget At Risk customers with personalized offers.
- Run win-back campaigns for Lost Customers.
- Engage Potential Loyalists through targeted promotions.
